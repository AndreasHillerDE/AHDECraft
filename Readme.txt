The AHDECraft Mod is a mod that contains the following things:

-   Conversion Ball which converts items and blocks into other items or blocks inside a crafting table or the 2x2 grid of the inventory and with the conversion wand you can change blocks inside the world.

-   The are new ores such as Red Coal, Tin, Silver and many other - even nether ores including vanilla ores

-   Also included are building blocks of these ores

-   New powerful tools and weapons and much more.

This is an early alpha version of this mod. Do not expect too much for now!
